Why This Exists

Most LLM Apps
Trust the Prompt Too Early

Standard WAFs and model-level guardrails fail because they can't understand the semantic meaning of LLM prompts or are easily bypassed by encoded, smuggled, or multi-step attacks.

01
⚡

Prompt Injection

Instruction overrides, jailbreaks, encoded payloads, and smuggled content hijack model behavior across text, images, and PDFs.

02
🔓

Session Abuse

Multi-step APT-style attacks exploit session context. Repeat offenders escalate risk without per-session tracking and cooldown controls.

03
🕵️

PII Leakage

Aadhaar, PAN, emails, and phone numbers surface verbatim in raw LLM responses without egress redaction.

Feature Highlights

Everything You Need to
Secure Your AI Stack

🛡️

Prompt & Input Defense

Screens text before provider execution. Detects instruction overrides, jailbreaks, encoded payloads, and smuggled content across text, OCR images, and PDFs.

🎯

Explainable Threat Scoring

Each request gets a risk level (GREEN/AMBER/RED), numeric threat score, action taken, score breakdown, and matched attack signals.

🔒

Session-Aware Abuse Control

Tracks risky behavior across requests, escalates repeat offenders, applies cooldown windows, and returns 429 when sessions are locked.

⚙️

Policy-Driven Guardrails

Loads policy from YAML, watches changes at runtime, builds system prompts with active guardrails, classifies intent before the provider sees it.

🔌

Multi-Provider Model Routing

Routes requests through OpenAI, Claude, Gemini, or Ollama. Provider can come from policy or be overridden per request.

🔏

Egress Protection

Before a response is returned, the middleware redacts PAN, Aadhaar, email addresses, and phone numbers. Redaction counts feed into telemetry and dashboards.

🧠

Adaptive Defense

Reads attack reports, ranks attack families, generates runtime signatures from indicators, recommends response playbooks, and applies changes into policy overlays. Covers indirect injection, tool execution, memory poisoning, moderation evasion, and encoded payloads.

📊

Live Operator Dashboard

Real-time React dashboard with threat charts, attack feed, session watchlist, red-team scenario launcher, adaptive-defense simulation, PII redaction totals, and before/after sanitization view.
